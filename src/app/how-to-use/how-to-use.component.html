COMO USAR?

Mecanismo de busca:

As buscas podem ser realizadas utilizando o teclado ABNT-2 onde os caracteres latino serão trocados pelos respectivos caracteres grego de acordo com o layout do teclado grego moderno, as buscas também podem ser feitas utilizando o teclado grego nativo

Layout do teclado:

Caso haja alguma dificuldade, ao clicar no teclado no canto direto do campo de busca, o usuario terá acesso a um teclado virtual.
O layout do teclado virtual é o mesmo layout utilizado para efetuar a troca de caracteres.

Zoom na versão Desktop:

Para os usuários Desktop, ao passar o mouse sobre a palavra uma função zoom-in é ativada para auxiliar na visualização dos diacriticos.

Texto original:

As palavras são representadas no dicionário da seguinte forma: palavra, declinação, classe, gênero, etimologia e significado
Algumas palavras apresentam infleccção e observação.
